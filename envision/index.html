Envision Projects

IEEE NITK Student Branch's Virtual Project Expo

VTOL

A prototype of a VTOL (Vertical Take Off and Landing) aircraft, with the primary goal of achieving maximum range and flight time for surveillance

Tiles Project

This project is based on the idea of generating electric power from the energy spent in the form of human walking, a renewable and continuous source, without using piezoelectric material.

Solenoid Engine

A solenoid engine is one of the viable alternatives to a motor used in Electric vehicles. Check out our project where we simulate and create a miniature Solenoid engine controlled by Arduino!

Smart Lock

Smart lock is an IoT device capable of automating the unlocking of doors. A piezoelectric sensor is used to a "secret" knocking pattern and bluetooth connectivity to allow remote access

Pong Game

An old classical game of the ball and walls which have single and multiplayer options and the player's main task is to prevent the collision of the ball on your backside wall.

Let Me Out

An AI-based maze game with the goal of learning a shortest (and most rewarding) path from the start cell to the goal cell in an environment full of obstacles, rewards, and opponents.

Home Automation

This project is aimed at automatic switching and control of electronic appliances with the detection of human presence. The basic idea is to save the amount of energy wasted on unnecessary usage of these appliances

Handwriting recognition and replication

This project aims to recognise handwritten text and recreate the same text in a different handwriting, thus essentially changing the handwriting of a given document

Gesture Control Bot

The project aims to make a bot that is controlled by the gesture of our hand. The transmitter circuit placed on the hand to senses hand movement and communicates with a receiver on the bot

Flame Detection Bot

A smart flame detection system is necessary in many industries, smart cities and even smart homes. A novel approach to this topic is our flame detection bot

E-Marketplace

This E-Marketplace is an attempt to connect people all over the NITK campus to help them buy and sell items, both academic as well as non-academic.

Cleaner Bot

This project is about fixing the small problems we have in our daily life so that our life can become more comfortable. The aim of this project is to be able to clean all the fine dust and sand particles in the house

Automated Plant Watering

A smart irrigation system which senses the moisture content of the soil and waters plants accordingly. It also keeps the moisture content of the soil within a specified range thus preventing overwatering.

2D Mapper

A bot which autonomously creates a 2D map of an unknown surrounding by navigating around the environment
